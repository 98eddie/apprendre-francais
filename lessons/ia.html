문법 포인트
1. 조동사 (Verbes modaux) — pouvoir, vouloir, devoir
영어의 can / want to / must 에 해당하는 프랑스어 조동사. 뒤에 항상 동사 원형(infinitif)이 온다.
pouvoir
[뿌부아흐]
~할 수 있다
(can)
vouloir
[불루아흐]
~하고 싶다
(want to)
devoir
[드부아흐]
~해야 한다
(must)
pouvoir 현재형 활용:
jepeux [쁘]
tupeux [쁘]
il / ellepeut [뻬]
nouspouvons [뿌봉]
vouspouvez [뿌베]
ils / ellespeuvent [뻬브]
Un medecin peut utiliser l'IA.
의사는 AI를 사용할 수 있다. (pouvoir + utiliser)
Je dois etudier le francais.
나는 프랑스어를 공부해야 한다. (devoir + etudier)
2. pour + infinitif — 목적 표현 (~하기 위해)
영어 "in order to"에 해당. pour + 동사 원형 구조.
Les startups levent des fonds pour developper de nouveaux produits.
스타트업들이 새 제품을 개발하기 위해 자금을 조달한다.
Un medecin utilise l'IA pour diagnostiquer les maladies.
의사가 질병을 진단하기 위해 AI를 사용한다.
J'etudie pour apprendre le francais.
나는 프랑스어를 배우기 위해 공부한다.

pour + infinitif는 영어 to-infinitive와 같은 역할이라 이미 직관적으로 느껴질 거야.
3. 부사 만들기 — 형용사 + ment
영어의 -ly 처럼 프랑스어는 형용사에 -ment를 붙여 부사를 만들어.

기본 규칙: 여성형 형용사 + ment
rapide → rapidement
빠른 → 빠르게
lent(e) → lentement
느린 → 느리게
facile → facilement
쉬운 → 쉽게
serieux/serieuse → serieusement
진지한 → 진지하게
비교급: plus + 부사 = 더 ~하게
diagnostiquer les maladies plus rapidement
질병을 더 빠르게 진단하다

오늘의 표현
Ca change tout !
[싸 샹쥬 뚜]
그게 모든 것을 바꾼다! / 완전 달라지는 거야!

ca = cela (그것). changer = 바꾸다. tout = 모든 것.
일상에서 "게임 체인저다!" 같은 뉘앙스로 많이 써.
L'IA ? Ca change tout dans la medecine.
(AI요? 의학에서 모든 걸 바꾸고 있어요.)

Ca ne change rien. [싸 느 샹쥬 히엥] = 아무것도 안 바뀌는 거야.
